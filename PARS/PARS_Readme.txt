This directory contains two sub-directories, FakePulsar and Pulsar respectively.
Both of these directories contain par files, used by Tempo2 to for pulsar timing
and other purposes. The par files are created by the CandidateParGenerator.py
script.

FakePulsar  -   This directory contains par files for fake pulsars. These
                par files are generated by the CandidateParGenerator.py script.
                The script creates fake pulsar parameters (period, DM, S/N), by
                either randomly sampling from user specified distributions,
                or by modelling distributions from user specified data. Once
                the parameters are generated, they are written to par files for
                input in to Tempo2.

Pulsar      -   This directory contains par files for real pulsars. These
                par files are generated by the CandidateParGenerator.py script.
                To create these files, the script has to read a valid ATNF pulsar
                catalog database file. It extracts the RA, DEC, period, and DM
                values from this file per source, and then stores them in a
                valid par file.

CandidateParGenerator   -   The python script used to generate par files.
